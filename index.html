Geometry / 3D Software Portfolio

Selected browser-based projects focused on computational geometry, 3D software, C++/WebAssembly, and manufacturing-adjacent workflows. These demos are small, but they reflect the kinds of systems I like building: interactive geometry tools, performance-sensitive algorithms, and software that connects mathematical and geometric logic to practical workflows.

MeshSlicer

STL slicing / 3D printing workflow

Browser-based STL slicer with TypeScript and C++/WebAssembly implementations, live plane-slice preview, and PNG layer-stack export.

This project is the most directly related to 3D printing workflows: it takes a triangle mesh, slices it into layers, and exposes the result interactively in the browser. It was built to show geometry-processing logic, browser-based 3D tooling, and the difference between a TypeScript implementation and a compiled C++/WASM backend.

Live demo GitHub

STL · slicing · C++/WASM · TypeScript · 3D printing · geometry processing

MeshViewerGL

WebGL2 mesh viewer / geometry inspection

Custom WebGL2 STL viewer with orbit/pan/zoom controls, shader modes, mesh overlays, and interactive geometry inspection.

This project focuses on lower-level 3D graphics and mesh visualization without relying on Three.js. It shows direct work with WebGL2, STL parsing and visualization, camera controls, rendering modes, and the kind of interactive tooling needed to inspect and debug geometric data.

Live demo GitHub

WebGL2 · STL viewer · 3D graphics · mesh visualization · interactive geometry

BoardRoute

C++/WASM routing / constrained path planning

C++/WebAssembly A* routing demo for constrained 2D board geometry, with clearance, bend, and via-style costs.

This project demonstrates algorithmic path planning over geometric constraints. The routing core is compiled to WebAssembly and exposed through a browser UI, showing how compiled C++ logic can drive an interactive frontend for search, routing, and constraint-based layout problems.

Live demo GitHub

C++/WASM · A* · path planning · geometry constraints · routing · browser visualization
